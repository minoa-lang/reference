# Constructing expressions
```
<construct-expr> := <tuple-expr>
                  | <array-expr>
                  | <struct-expr>
                  | <vectorized-expr>
                  | <with-expr>
```

A constructing expressions is a group of expression that produce a new instance of a given type.
This consists out of a group of expressions which can be used in with an [in-place operator].



[in-place operator]: ../operators/special-operators.md#in-place-operator-
